# 智能旅游推荐系统 python1035

## 项目概述

本项目是基于Python的Django框架和Vue.js的前端技术构建的智能旅游推荐系统。系统致力于为用户提供一站式的旅游信息服务，包括景点推荐、旅游资讯、行程分享等。

## 技术栈

- 后端：Python + Django
- 前端：Vue.js
- 数据库：MySQL

## 功能模块

### 用户端

- 首页：展示系统概览和热门旅游推荐
- 个人中心：用户信息管理、行程管理等
- 旅游资讯管理：浏览和搜索旅游资讯
- 景点信息管理：查看景点详细信息
- 行程分享管理：分享和查看他人行程

### 管理端

- 用户管理：管理用户信息和权限
- 景点信息管理：添加、编辑和删除景点信息
- 景点分类管理：分类管理景点信息
- 酒店信息管理：管理酒店信息
- 交流论坛：系统用户交流互动区域
- 系统管理：系统设置和运营管理

## 系统角色

- 普通用户：浏览信息、管理个人行程等
- 管理员：系统运营管理、用户管理等

## 运行环境

- Python环境
- MySQL数据库环境
- Node.js环境（前端）

## 部署步骤

1. 准备Python运行环境
2. 安装依赖库
3. 配置MySQL数据库
4. 部署前端Vue项目
5. 运行后端Django项目
